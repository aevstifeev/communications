# Screens

Base:
- `index.html` — базовая kanban-доска Kaiten с shell-структурой, панелями и drawer-ом.

Planned communication-related screens:
- карточка встречи;
- календарь пространства;
- интеграция встречи в карточку задачи;
- сценарий планирования встречи из задачи;
- сценарий входа во встречу из календаря;
- сценарий работы с участниками.

Правило:
- отдельная сущность или самостоятельный UX-объект получает собственный экран или собственную concept-страницу.
