# Project Context

Продукт:
- Kaiten

Цель текущего репозитория:
- собирать интерактивные прототипы новых UX-сценариев без доступа к основной кодовой базе продукта;
- использовать текущий интерфейс Kaiten как контекст и оболочку для встраивания новых функций.

Основные типы задач:
- интеграция модуля коммуникации: чаты, видеоконференции, участники;
- переработка карточек, drawer-ов и связанных сценариев;
- быстрые концепты для обсуждения с коллегами;
- параллельные версии экранов и пользовательских flow.

Текущее состояние:
- есть базовый shell с доской, левой и правой панелями;
- подключены дизайн-токены из Figma через CSS variables;
- проект используется как основа для дальнейших концепций.
