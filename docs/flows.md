# Flows

Основные flow для будущей проработки:

1. Планирование встречи из карточки задачи.
2. Переход во встречу из календаря пространства.
3. Работа с участниками до, во время и после встречи.
4. Коммуникация по задаче через чат или видеозвонок.

Правило работы:
- сначала можно проектировать элементы отдельно;
- после появления базовых элементов собираем их в сквозные flow;
- для альтернативных flow используем отдельные concept-страницы или отдельные ветки.
