# Decisions

## Current base
- Базовая точка входа — kanban-доска Kaiten.
- Базовая страница должна оставаться рабочей и стабильной.
- Дизайн-токены из Figma подключены как источник истины через `styles/tokens.css`.

## File organization
- Общие стили лежат в `styles/`.
- Общая интерактивная логика лежит в `scripts/`.
- Контекст и проектные договорённости лежат в `docs/`.
- Новые концепции и вариации раскладываются по `concepts/`.

## Working rule
- Не перегружаем `index.html` всеми экспериментами сразу.
- Новые продуктовые гипотезы оформляем как отдельные concept-направления.
