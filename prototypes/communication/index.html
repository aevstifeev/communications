Communication Module

Хаб для активных версий модуля коммуникации

Здесь собраны основные варианты для сравнения: базовая контрольная точка, распределённая версия и централизованный модуль коммуникации. Архивный drawer-паттерн сохранён отдельно и не входит в основной контур демонстрации.
