# Centralized Communication Prototype

Рабочая папка второго прототипа коммуникации.

## Что это

Централизованный модуль коммуникации как самостоятельный слой системы.

## Как используем

- Здесь развиваем вторую активную версию.
- `../distributed/` остается гипотезой про распределённую коммуникацию.
- `../archive-context-drawer/` остается архивным референсом и не смешивается с модульной концепцией.
