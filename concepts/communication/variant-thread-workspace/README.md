# Variant B - Thread Workspace

Рабочая папка второго прототипа коммуникации.

## Что это

Централизованный модуль коммуникации как самостоятельный слой системы.

## Как используем

- Здесь развиваем версию 2 в текущем чате.
- `variant-inline-panel/` остается первой гипотезой про встроенную коммуникацию.
- `variant-context-drawer/` остается архивным референсом и не смешивается с модульной концепцией.
