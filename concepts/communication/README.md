# Communication Concepts

Здесь сохраняется исследовательский слой по теме коммуникации.

## Что находится здесь сейчас

- отдельные концептуальные направления модуля коммуникации;
- legacy-структура, из которой выросли текущие publish-ready прототипы;
- дополнительные ветки исследования вроде `meeting-card/`, `space-calendar/` и `task-integration/`.

## Где теперь активные веб-прототипы

Активный набор для GitHub и Vercel перенесён в:

- `prototypes/communication/base/`
- `prototypes/communication/distributed/`
- `prototypes/communication/centralized/`
- `prototypes/communication/index.html`

## Роль этой папки

- хранить продуктовые направления и заметки, которые не обязаны быть публичной точкой входа;
- не смешивать исследовательские наброски с основным runtime-контуром репозитория;
- сохранять исторический контекст без захламления корневой структуры.
