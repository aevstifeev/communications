# Kaiten Prototype Workspace

Прототип интерфейсов Kaiten для проработки новых продуктовых сценариев.

Текущие принципы работы:
- `index.html` — стабильная базовая точка входа.
- `styles/` — токены и общие стили проекта.
- `scripts/` — общая интерактивная логика shell-уровня.
- `docs/` — контекст проекта, решения, сценарии и todo.
- `concepts/` — отдельные направления и варианты концепций.

Базовое правило:
- новые идеи не встраиваем хаотично в одну страницу;
- сначала решаем, это изменение `base`, отдельный `concept` или отдельный `flow`.
